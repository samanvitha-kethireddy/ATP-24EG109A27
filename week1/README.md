# Assignments Repository

## Overview
This repository contains solutions to JavaScript programming assignments.  
Each file demonstrates a basic concept like arrays, functions, and comparisons.

## Files Included
- `arrayAsArgument.js` → Passing arrays as function arguments
- `emp.js` → Employee data handling
- `functions.js` → Collection of simple functions
- `greatestOf2.js` → Greatest of two numbers
- `greatestOf3.js` → Greatest of three numbers
- `indexOfArrayElement.js` → Search element in array
- `sumAndMin.js` → Sum and minimum of array elements

## How to Run
Use Node.js:
node filename.js
