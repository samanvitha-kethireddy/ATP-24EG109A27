#blog app

MERN application of a Blog App which has users and authors.

Authors can:
  - write articles
  - edit articles
  - delete articles
  - view users' comments on their articles

Users can:
   - view articles
   - read articles
   - add comments

Admins can:
  - view active status of all users
  - block user/admin profiles
  - unblock user/admin profiles
