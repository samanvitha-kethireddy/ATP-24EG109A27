# Web Projects

## HTML & CSS
- `billsAndMarks/` → Tables for bills and student marks  
- `catPage/` → Themed page with image, headings, and cards  
- `cityGuide/` → City info layout with circular images and columns  
- `coloredTable/` → Styled table with alternating row colors  
- `hulkCards/` → Card layout with navbar and profile cards  
- `webpageLayout/` → Simple two-section layout with buttons  

Run:
 Use Live Server.

- `productsCardLayout` → Product listing (cards with descriptions)  
- `profilesCardLayout` → User profiles (grid with images/emails)  
- `usersCounter` → User management (list with add user + count)

npm install
