# JavaScript Assignments

## Overview
This repository contains JavaScript assignments demonstrating arrays, functions, objects, classes, and ES6 features.  
Each file focuses on a specific concept or use case.

## Files Included
- `sumOfNumbersFunction.js` → Sum any number of arguments using rest parameters
- `deepCopy.js` → Deep copy object using `structuredClone`
- `eCommerceCart.js` → Shopping cart operations with filter, map, reduce, find, findIndex
- `examEvaluation.js` → Timed messages using `setTimeout`
- `fruitArray.js` → Copy and extend arrays with spread operator
- `libraryManagement.js` → Book class with borrow/return and info methods
- `objectCopy.js` → Object copy and extension using spread operator
- `productAPI.js` → Mini Express router for product CRUD APIs
- `shallowCopy.js` → Shallow copy example showing reference behavior
- `temperatureAnalyzer.js` → Array operations on temperature data
- `toDoList` → Todo List app combining validation, task management, and demo usage

## How to Run
Use Node.js to execute any file:
node filename.js
