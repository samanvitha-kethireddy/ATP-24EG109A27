# HTML & CSS Practice

## Overview
This repo contains small practice examples for learning HTML and CSS:
- Nested unordered lists
- Basic page with headings, text, and links
- A styled Hello Kitty profile page

## Structure
- `NestedUnorderedList.html` → Nested snack list
- `basicPage.html` → Simple page with sections and a link
- `helloKitty/`  
  - `catIndex.html` → Hello Kitty profile  
  - `catStyle.css` → Stylesheet

## Usage
Open any `.html` file in a browser.
